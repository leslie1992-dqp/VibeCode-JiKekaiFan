# 全量数据库表 Wiki

## 文件位置

- 主文件：`wiki/database/tables.md`。
- 可选：按域拆分 `wiki/database/tables-user.md` 等，并在 `tables.md` 顶部做目录索引。

## 内容要求

1. **表清单**：列出当前仓库迁移脚本所反映的 **全部表**（表名、一句话用途）。
2. **每张表一节**：包含
   - 表名（与数据库一致）
   - 主键、唯一约束、重要索引（文字描述即可）
   - 字段表：列名、类型、可空、默认值、说明
3. **与代码映射**：注明对应实体类名（如 `UserEntity`）与 Mapper 包路径（可选）。

## 同步规则

- DDL 或 Flyway/Liquibase 脚本变更后，**同一迭代**内更新 `tables.md`。
- 删除表时，从清单中移除或标记「已废弃」并注明下线版本/日期。

## 与 ER 图分工

- `tables.md`：**字段级真相来源**。
- `er-diagram.md`：**关系与实体级概览**。两者冲突时，以迁移脚本为准，然后修正文档。

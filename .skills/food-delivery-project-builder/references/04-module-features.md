# 模块功能清单

## user 模块

- 手机号注册与密码注册
- 登录并签发 JWT
- 查询当前用户信息
- 地址增删改查
- 默认地址切换

## merchant 模块

- 商家列表分页与关键字搜索
- 商家详情查询
- 商家营业状态校验

## product 模块

- 按商家查询分类列表
- 按商家/分类/关键字查询商品列表
- 商品详情查询
- 商品库存与上下架状态校验

## cart 模块

- 添加商品到购物车
- 更新数量与勾选状态
- 删除购物车项
- 清空指定商家购物车
- 购物车价格汇总计算

## order 模块

- 订单预览试算
- 基于购物车项创建订单
- 订单详情查询
- 用户订单分页查询
- 订单取消（用户取消 + 超时取消）
- 订单状态流转控制

## coupon 模块

- 秒杀券列表与倒计时
- 秒杀请求受理
- 秒杀结果轮询
- 一人一单校验
- 秒杀库存预占与补偿

## payment 模块

- 创建模拟支付记录
- 处理模拟支付回调
- 按订单查询支付状态
- 支付回调幂等处理

## common 与 infrastructure 模块

- 统一返回结构封装
- 全局异常处理
- 请求日志与链路追踪 ID
- Redis 缓存工具能力
- Lua 执行封装
- Kafka 生产消费事件处理
- JWT 解析与鉴权过滤器
